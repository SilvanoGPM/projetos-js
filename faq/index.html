Perguntas frequentes:

Como ver os projetos?

Para visualizar os projetos você deve clicar no menu ao lado, isso abrirá uma sidebar para você escolher o projeto que deseja visualizar.

Caso queria visualizar o código fonte dos projetos, eles estão disponíveis neste respositório.

Como reportar bugs?

Caso ache algum bug em um projeto, você pode abrir um pull request no github ou tentar me contatar diretamente usando o email silvanosilvino1@gmail.com.

O que são os projetos?

Os projetos variam desde desafios até algo que eu somente estava com vontande de criar.
